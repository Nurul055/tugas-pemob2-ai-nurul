1.	Konsep Utama 
SmartSpend untuk mencatatan keuangan fokusnya adalah pada "Goal-Based Saving
"SmartSpend dibuat bukan cuma buat nyatet angka, tapi buat ngubah kebiasaan. Banyak orang nyatet pengeluaran tapi tetep boros. Dengan fitur Goal-Based Saving, aplikasi ini ngebantu user buat punya prioritas: mending jajan sekarang atau nabung buat beli laptop? Jadi ada unsur edukasi keuangan-nya."
Contoh: kita punya target nabung buat bayar publikasi jurnal atau beli laptop baru. Nah, web/aplikasi ini bakal nemenin kita sampai target itu 100%.


2.	Isi 7 Halaman
Ini rincian fungsi tiap halamannya dalam aplikasi SmartSpend:

•	Halaman Login (Pintu Masuk): User masukin akun. Penting karena data keuangan itu privasi.

•	Halaman Register (Bikin Akun): Tempat user baru daftar. Namanya tersimpan.

•	Halaman Beranda (Dashboard Keuangan): Ini rangkuman besarnya. Isinya saldo total, sisa uang hari ini, sama grafik simpel pengeluaran. Datanya berubah otomatis tiap ada transaksi baru.

•	Halaman Catat Transaksi (Input Data): Di sini ada form: Nama barang, nominal, sama kategori (Makan, Transport, atau Cicilan). Begitu klik simpan, datanya langsung meluncur ke Firebase.

•	Halaman Riwayat (Buku Besar): List semua transaksi kamu dari yang paling baru. Di sini bisa kelihatan kalau kamu lagi boros di bagian mana.

•	Halaman Target Tabungan (Fitur Unggulan): Ini halaman khusus buat bikin target. Misal: "Beli Laptop". Nanti ada gambar progres bar yang makin penuh seiring kamu masukin uang ke sana.

•	Halaman Kurs & Berita Ekonomi (REST API): Isinya info harga Dollar ke Rupiah hari ini atau berita bisnis. Jadi user nggak cuma nyatet uang, tapi tau info ekonominya juga.
